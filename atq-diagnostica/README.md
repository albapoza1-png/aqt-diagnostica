# ATQ Diagnosi — Abans Que Tanqui

Eina interna de diagnosi de risc per a comerços locals.

## Instal·lació local

```bash
npm install
npm run dev
```

## Publicar a Vercel

1. Puja aquest repo a GitHub
2. Ves a vercel.com → "Add New Project"
3. Connecta el repo → Deploy
4. Llest! Tens una URL pública

## Afegir al mòbil (PWA)

Un cop publicat a Vercel:
- **iPhone**: Safari → botó compartir → "Afegir a pantalla d'inici"
- **Android**: Chrome → menú (⋮) → "Afegir a pantalla d'inici"

L'app funciona offline un cop carregada.
